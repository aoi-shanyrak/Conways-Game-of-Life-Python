# Conway's Game of Life - Python Implementation

Проект реализует симуляцию "Игры Жизни" Конвея с возможностью сравнения производительности разных подходов.

## Описание

Симуляция клеточного автомата "Игра Жизни" с двумя реализациями:
1. **NumPy** - оптимизированная версия с использованием векторных операций
2. **Стандартные списки Python** - простая реализация для сравнения производительности

## Требования

- Python 3.8+
- NumPy
- Matplotlib

Установите зависимости:
```bash
pip install numpy matplotlib
```

## Настройка конфигурации

Основные параметры настраиваются в файле `config.py`:

```python
# Основные параметры
FIELD_SIZE = 256      # Размер поля (256x256 клеток)
STEPS = 64           # Количество шагов симуляции
DENSITY_VALUE = 0.25 # Плотность начального заполнения (25%)

# Выбор реализации
USE_NUMPY = True    # True - использовать NumPy, False - стандартные списки
USE_NUMPY_UNEFFECTIVE = False  # True - неэффективная реализация NumPy (для сравнения)

# Анимация
ANIMATION = True     # Показывать анимацию
INTERVAL_FOR_ANI = 50  # Интервал между кадрами (мс)
SAVE_GIF_ANI = False  # Сохранять анимацию в GIF
SAVE_GIF_ANI_PATH = "gifs"  # Папка для сохранения GIF

# Отладка
VERBOSE = True       # Вывод дополнительной информации
```

## Производительность

Проект позволяет сравнить производительность:
- **NumPy с оптимизированными операциями** - быстрее всего
- **NumPy с поэлементной обработкой** - для сравнения
- **Стандартные списки Python** - самая медленная версия

Время выполнения выводится в консоль при `VERBOSE = True`.

## Анимация

Анимация создаётся с помощью Matplotlib и показывает:
- Эволюцию клеточного поля по шагам
- Чёрные клетки - "живые"
- Белые клетки - "мёртвые"

### Управление анимацией:
- Автоматическое проигрывание
- Возможность сохранения в GIF
- Настройка скорости анимации через `INTERVAL_FOR_ANI`

## Правила игры

1. **Выживание:** Клетка с 2 или 3 соседями остаётся живой
2. **Смерть:** Клетка с менее чем 2 или более чем 3 соседями умирает
3. **Рождение:** Мёртвая клетка с ровно 3 соседями становится живой

## Отладка

При включённом `VERBOSE = True` в консоль выводится:
- Параметры симуляции
- Используемый файл поля
- Время выполнения
- Информация о созданных полях
